# Changelog

All notable changes to the BC Code Intelligence extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-01-15

### Added
- **New Specialists**: Added Lena Pipe (CI/CD pipelines & DevOps) and Victor Versioning (BC version migrations)
- **VSCode Extension Knowledge**: Created comprehensive `vscode-extension.md` guide in chris-config domain explaining extension features, settings, and usage
- Enhanced CodeLens patterns with 169 mappings covering all specialists
- Workflow examples updated with 12 embedded workflows including AppSource submission, security audit, performance audit, and more

### Changed
- Updated to bc-code-intelligence-mcp v1.7.0
- Improved specialist discovery and workflow routing
- Updated chat participants and agents list to reflect current specialist roster (17 total)

### Removed
- **Casey Copilot specialist** - functionality consolidated into other specialists
- Cleaned up obsolete references in source code and documentation

### Fixed
- Corrected specialist list across all extension components
- Updated default specialist recommendations to use Sam Coder and Chris Config
- Fixed common names list in specialist discovery

## [0.1.11] - Previous Release

Initial releases with core specialist functionality and MCP integration.
